# L Rank of a constructible set
 
>[!definition]
> If $x\in L$, let $\rho(x)$ be the least $\beta$ such that $x\in L(\beta+1)$

>[!lemma]
>For any $\alpha$,
>- $L(\alpha) = \{ x\in L: \rho(x) < \alpha \}$
>- $\rho(\alpha)=\alpha$
>- $L(\alpha) \cap \mathbf{ON} = \alpha$
